Esta carpeta contiene el Manual de Administrador de TechnoStore, la plataforma web de comercio electrónico orientada a la venta de componentes y equipos informáticos.
El manual está dirigido a administradores técnicos y personal de sistemas que gestionan las 3 máquinas virtuales (web, aplicación, base de datos) a través de SSH y herramientas de monitorización.
